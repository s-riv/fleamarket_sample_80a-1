# README

## アプリケーション概要
* フリーマーケット用のWebアプリです。５人のチームでアジャイル開発を行いました。

## 機能一覧
- ユーザー登録、ログイン機能（ウィザード形式）
- 商品出品機能
- 商品購入機能
- クレジットカード登録機能（PayJp）
